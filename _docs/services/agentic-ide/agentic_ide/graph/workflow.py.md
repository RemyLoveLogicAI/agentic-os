## Overview

This module is responsible for assembling the LangGraph workflow used by the Agentic IDE command loop. It exposes two top-level functions: build_workflow, which constructs the graph structure using StateGraph, WorkflowState and node definitions, and compile_workflow, which returns a compiled, invocable graph (ready to run) from that structure. The implementation relies on the langgraph library primitives (StateGraph and END) and the agentic_ide.graph nodes and WorkflowState definitions to define states and transitions.
