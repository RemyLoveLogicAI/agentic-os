## Overview

This module defines a set of standalone node functions that operate on a WorkflowState to implement steps of a LangGraph workflow. The functions cover intent and risk classification, conditional routing based on risk, asynchronous execution of selected tools, parking actions for human approval, rejecting problematic commands, and final verification and evidence capture. Together they represent the core node-level transforms used to progress, pause, or terminate workflow execution in response to command analysis and tool outcomes.

## Maintenance Notes

- Classify raw commands to determine intent and associated risk level (classify).
- Choose the next workflow node based on the assessed risk (route).
- Execute the selected tool asynchronously and record the result (execute).
