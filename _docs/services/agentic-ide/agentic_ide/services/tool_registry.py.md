### Description

ToolRegistry manages a collection of tools by name, allowing callers to register tools, retrieve them, list what is available, and invoke a tool asynchronously.

Within the services/agentic-ide module, ToolRegistry provides centralized management and invocation of tools, enabling other parts of the system to discover and execute tools by name. It acts as the single place for tool lifecycle operations used by the agentic IDE services.

Responsibilities:
- Initialize and maintain the internal registry of tools on construction.
- Register new tools so they can be discovered and used by name.
- Retrieve a registered tool by its identifier.
- Provide a list of currently available tools.
- Invoke a registered tool asynchronously and return its result.
