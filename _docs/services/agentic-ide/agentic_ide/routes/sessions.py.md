This module defines the HTTP route handlers related to session lifecycle and agent command execution for an agentic IDE service. It exposes asynchronous functions to create and retrieve sessions, execute agent actions, and fetch session event history. The handlers rely on domain models for requests and responses, a session store service for persistence, and a workflow compiler component to prepare agent workflows.

The implementation is organized as FastAPI route handlers (via APIRouter) that work with typed models such as ActRequest/ActResponse and SessionResponse, and they coordinate between the persistent session store and the workflow compilation logic to manage and drive agent sessions and their recorded events.
