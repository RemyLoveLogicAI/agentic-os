## Overview

This test module contains asynchronous integration tests for the Agentic IDE FastAPI application. It uses pytest together with httpx's ASGITransport and AsyncClient to exercise the app instance imported from agentic_ide.main. The file defines a reusable async client fixture and a suite of tests that verify basic app behavior, session management endpoints, various action endpoints (including echo, status checks, approval flows, and rejection handling), and endpoint behavior when a session is not found, as well as history retrieval.
