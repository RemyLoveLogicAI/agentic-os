## Overview

This directory contains the implementation and tests for the Agentic IDE service. At the top-level there are no standalone files; instead the implementation is organized into the agentic_ide package which provides four primary modules (package initializer, config.py implementing application configuration with pydantic-settings integration, main.py as the FastAPI application entry point, and data models) alongside subpackages that implement the LangGraph workflow, HTTP routes, and runtime services. These modules implement the core service surface for the Agentic IDE and establish configuration and application startup behavior.

A parallel tests/ subdirectory contains unit and integration tests for the package. The tests focus on the Agentic IDE's LangGraph workflow (classification and routing behaviors) and the FastAPI application (asynchronous endpoints and routing). Together, the package and tests provide a cohesive code area for developing, running, and validating the Agentic IDE service components: configuration and startup, runtime services and graph workflow implementation, HTTP route handling, and automated verification through the test suite.


### File Organization

Code is organized as a package (agentic_ide/) containing the service implementation modules and subpackages for workflow, routes, and runtime services; tests are colocated in a separate tests/ subdirectory to isolate unit and integration test code from runtime implementation.
