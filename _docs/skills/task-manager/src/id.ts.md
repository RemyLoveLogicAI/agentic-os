## Overview

Module exports generateTaskId(): string which returns IDs in the form `tsk_${randomSegment(8)}`. It defines ALPHABET = "0123456789abcdefghijklmnopqrstuvwxyz" and an internal helper randomSegment(length: number) that builds a string by selecting characters using Math.random() and Math.floor(). The module is dependency-free and lightweight.

Intended for places that need short, human-readable task identifiers. Randomness is non-cryptographic; for reduced collision risk or stronger unpredictability, increase the segment length or use a cryptographically secure RNG.

## Architecture Notes

- Small, functional module with no side effects: defines a constant alphabet, an internal helper randomSegment(length: number): string, and an exported generateTaskId(): string.
- Uses Math.random() for sampling characters (non-cryptographic). If reproducible or cryptographically-secure IDs are required, replace Math.random() with an injected RNG or crypto API.
- Prefixing with 'tsk_' separates these IDs from other identifiers and centralizes ID format in one place for easy change.

## Functions and Classes


#### function randomSegment

![Type: Sync](https://img.shields.io/badge/Type-Sync-green)

### Signature

```typescript
function randomSegment(length: number): string
```

### Description

Generates and returns a string of the requested length composed of characters randomly selected from a module/global ALPHABET array.


The function initializes an empty result string then iterates length times. On each iteration it picks a random index using Math.floor(Math.random() * ALPHABET.length) and appends the character at that index from ALPHABET to the result; finally it returns the accumulated string. ALPHABET is referenced but not defined within this snippet, so it must exist in the surrounding module scope.

### Parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `length` | `number` | ✅ | Number of characters to generate for the resulting string.
<br>**Constraints:** Should be a non-negative integer (function does not validate this), If zero, an empty string is returned |

### Returns

**Type:** `string`

A string whose length equals the provided length parameter, composed of characters drawn from ALPHABET.


**Possible Values:**

- A string of length `length` consisting of characters from ALPHABET
- Empty string if length is 0

### Complexity

O(n) where n is the length parameter

### Notes

- Relies on a module/global ALPHABET array or string being defined in scope; if ALPHABET is empty this will produce undefined-index accesses resulting in 'undefined' appended.
- Uses Math.random(), which is not cryptographically secure; do not use for security-sensitive ID generation.
- Function does not perform input validation (e.g., non-integer or negative values).
