## Overview

This directory serves as the documentation-level index for skill implementations and their associated source documentation. At the root level there are no files; instead the directory's primary function is to group per-skill documentation subdirectories. Each subdirectory is intended to contain a documentation index and pointers to the skill's source code and type definitions.

Currently the only subdirectory is task-manager/, which contains the documentation-level source index and a TypeScript source submodule for the task-manager skill. That subdirectory documents the skill implementation and points to a src/ subdirectory that holds the TypeScript modules (including an index module and type definitions) and a UI submodule. Together, the top-level skills directory and its subdirectories provide a navigable structure so developers can find both high-level documentation and the corresponding TypeScript source for each skill.


### File Organization

This directory is organized hierarchically: the root contains no files and acts as a container for per-skill subdirectories. Each subdirectory (for example task-manager/) contains the documentation index and a src/ subfolder with the skill's TypeScript source and related modules. This keeps documentation and implementation grouped by skill.
