# Agentic OS Spec Pack

## Narrative

This pack describes a voice-native, governance-gated agent operating system for the Agentic Web.

It is designed to do one thing extremely well: turn human intent into safe, auditable, reproducible action.

## Included Documents

- `master-spec.md` — the canonical system contract
- `deployment-blueprint.md` — how the system is staged and deployed
- `phase-0-build-plan.md` — the first shippable slice
- `repo-layout.md` — the recommended folder topology
